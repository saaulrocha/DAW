Las listas en HTML

Las listas son una forma esencial de estructurar la información en una página web

Tipos de listas

Existen tres tipos principales de listas en HTML

  • Unordered lists (<ul>) - listas sin un orden específico
  • Ordered lists (<ol>) - listas numeradas o con letras
  • Description lists (<dl>) - listas con términos y definiciones

Unordered lists

Las listas desordenadas usan varios tipos de discos:

  • Disco
  • Círculo
  • Cuadrado

Ordered lists

Las listas ordenadas pueden usar números, letras o números romanos

  1. Números
  2. Letras mayúsculas
  3. Letras minúsculas
  4. Números romanos en mayúsculas
  5. Números romanos en mayúsculas
